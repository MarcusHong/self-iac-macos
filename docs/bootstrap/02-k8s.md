# K8s 워크로드

OrbStack K8s 위에 올라가는 워크로드들. 호스트 데이터 레이어에 의존한다.

## 실행

```bash
./bootstrap/02-k8s.sh
```

## 설치 순서

| 순서 | 컴포넌트 | 네임스페이스 | 용도 |
|:---:|---------|:----------:|------|
| 1 | Traefik | traefik | Ingress + TLS 종단 |
| 2 | GitLab CE | gitlab | Git + CI/CD + Registry |
| 3 | Flux | flux-system | GitOps 동기화 |

## 1. Traefik

K8s Ingress Controller. Tailscale 인증서를 사용하여 HTTPS를 종단한다.

| 포트 | 용도 |
|:----:|------|
| 80 | HTTP |
| 443 | HTTPS |

TLS Secret은 `traefik` 네임스페이스의 `tailscale-tls`에 저장된다.

## 2. GitLab CE

Helm chart로 배포. 모든 stateful 컴포넌트는 호스트를 사용한다.

**내장 서브차트 비활성화 목록:**
- PostgreSQL → 호스트 사용
- Redis → 호스트 Valkey 사용
- MinIO → 호스트 사용
- Nginx Ingress → Traefik 사용
- cert-manager → Tailscale TLS 사용
- Prometheus → 비활성화

**활성화된 컴포넌트:**
- Webservice (Puma 1 worker)
- Sidekiq
- Gitaly (50GB PV)
- Runner (K8s executor, privileged, MinIO 캐시)
- Container Registry (주간 GC)
- Toolbox (일일 자동 백업)

**SMTP (Gmail):**

`SMTP_USER`와 `SMTP_PASSWORD` 환경변수를 설정하면 Gmail SMTP로 메일을 발송한다.
Google 앱 비밀번호 필요 (https://myaccount.google.com/apppasswords).

```bash
export SMTP_USER="your@gmail.com"
export SMTP_PASSWORD="앱비밀번호"
```

**자동화 스케줄:**

| 작업 | 스케줄 | 설명 |
|------|--------|------|
| GitLab 백업 | 매일 04:00 | MinIO `gitlab-backups` 버킷에 저장 |
| Registry GC | 매주 일 05:00 | 미사용 이미지 레이어 정리 |
| TLS 인증서 갱신 | 격월 1일 03:00 | Tailscale 인증서 자동 갱신 |

**Runner 캐시:**

CI/CD 빌드 캐시를 MinIO `gitlab-runner-cache` 버킷에 저장하여 빌드 속도를 개선한다.

**호스트 연결:** K8s에서 호스트 서비스에 `host.docker.internal`로 접근한다 (OrbStack 제공).

**초기 root 비밀번호:**

```bash
kubectl get secret gitlab-gitlab-initial-root-password \
  -n gitlab -ojsonpath='{.data.password}' | base64 -d; echo
```

스크립트가 자동으로 OpenBao에도 저장한다.

## 3. Flux

GitLab 셀프호스팅 인스턴스에 연결된다.

```
flux bootstrap gitlab \
  --hostname="$GITLAB_HOST" \
  --owner=infra \
  --repository=infra \
  --path=clusters/mac-mini
```

**필요한 GitLab 설정 (스크립트 실행 중 안내됨):**

1. https://<tailscale-hostname> 접속 → root 비밀번호 변경
2. User Settings → Access Tokens
   - Name: `flux-bootstrap`
   - Scopes: `api`, `read_repository`, `write_repository`
3. `infra` 그룹 생성 (또는 Flux가 자동 생성)

**동기화 경로:** `clusters/mac-mini/`

이후 이 경로에 매니페스트를 push하면 Flux가 자동 배포한다.
